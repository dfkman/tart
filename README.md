tart-OLD
============

Overview:
----------

THIS IS THE ARCHIVED VERSION, BASED ON THE ORIGINAL PROTOTYPE.

tart is a C (ncurses) based image editor for NIX. 

Features (planned):
- Open jpg, png and "view" them
- Edit said files
- Save them to disk
